# Passion Finder (placeholding name) -- CSC207 Final Project Team 8

### Summary:
Personalized University of Toronto Course Explorer for students of various backgrounds. The purpose is to provide University of Toronto students with a user-friendly tool to find courses they care about, learn about the course’s basic info, and eventually optimize their university journey planning.

### User Story:
1. As a user, I want the app to securely save my API Key, ‘courses taken’ and the last submitted ‘interests’. (Andrew)
Create user database (architecture)

2. As a student who is unsure of my interests, I want the app to have an interest survey that will recommend me keywords to use. (Izay)

3. As a user, I want the app to recommend courses based on my courses taken and the submitted interests. (Hongyi)

4. As a student, I want to be able to click on a recommended course to see the course description. (Harry)
Show description

5. As a user, I want to be able to see why a course was recommended for me. (Arthur)
Connection between course description and my interests, short LLM response, 1-2 sentences. 


### APIs Used:
Google Developer API: Gemini 2.5 Flash; Free of Charge. 
Main Service Provided: Web scraping, key words (interests) matching w/ course descriptions.
